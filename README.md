# Smart Campus Information System

A Python-based mini project that integrates multiple academic management modules into one unified application.  
Developed as part of the **Python Programming Laboratory (Program 9)** at Dayananda Sagar College of Engineering.

---

## 🚀 Features

- **Student Registration**: Register students with unique IDs, evaluate grades, and assign performance remarks.
- **Course Enrollment**: Manage up to 5 courses per student with credit validation.
- **Student Records**: Store and display student details from a CSV file.
- **Search & Sort**: Sort student IDs and search for specific IDs.
- **Fee Calculation**: Compute total fees (tuition, hostel, transport).
- **File Handling**: Read and display student records from `students.csv`.
- **Directory Scan**: Scan folders and handle empty directory errors with custom exceptions.
- **Performance Analytics**:
  - Statistical summary using Pandas
  - NumPy analysis (mean, median, standard deviation)
  - Highest performer(s) and Top 3 performers
  - Grade distribution
  - Visualizations with Matplotlib (bar charts, optional pie charts)
